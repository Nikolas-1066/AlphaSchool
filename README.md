# AlphaSchool
Распакуйте библиотеки в libraries
- Подключение Модулей к Ардуино
- На плате PN532 включите первый ползунок в положение 1
-  Connect
- PN532
	- GND ----- GND
	- VCC ----- 3.3V
	- SDA ----- A4
	- SCL ----- A5
- Servo
	- GND ----- Коричневый
	- 5V ------ Красный(Центральный)
	- 6pin ---- Оранжевый
- Time_Module
	- VCC ----- 5V
	- GND ----- GND
	- CLK ----- A0
	- DAT ----- A1
	- RST ----- A2
- LCD_I2C
	- GND ----- GND
	- VCC ----- 5V
	- SDA ----- A4
	- SCL ----- A5
- RGB_Lenta
	- +5V ----- 5V
	- GND ----- GND
	- Din ----- 7
- HC-06
	- +5V ----- 3.3V
	- GND ----- GND
	- TX  ----- 10
	- RX  ----- 11
